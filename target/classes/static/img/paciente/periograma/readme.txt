Tamanho padrão: 840 x 160 (taxa de aspecto 21:4)
Sugestão de biblioteca para uso: Snap.svg [http://snapsvg.io/]

Padrão usado nos IDs dos objetos SVG:

[face]_[numero]   <- identificação do dente
  [face]_[numero]_dente   <- dente
    [face]_[numero]_raiz   <- raiz do dente
    [face]_[numero]_coroa   <- coroa do dente
  [face]_[numero]_implante   <- implante (substitui o dente)
    [face]_[numero]_implante_parafuso   <- parafuso do implante
    [face]_[numero]_implante_dente   <- coroa do dente do implante
  ?[face]_[numero]_furca   <- furca do dente (em dentes que possuem somente uma furca)
    [face]_[numero]_furca_1   <- nível 1 da furca
    [face]_[numero]_furca_2   <- nível 2 da furca
    [face]_[numero]_furca_3   <- nível 3 da furca
  ?[face]_[numero]_furca_esquerda   <- furca esquerda do dente
    [face]_[numero]_furca_esquerda_1   <- nível 1 da furca esquerda
    [face]_[numero]_furca_esquerda_2   <- nível 2 da furca esquerda
    [face]_[numero]_furca_esquerda_3   <- nível 3 da furca esquerda
  ?[face]_[numero]_furca_direita   <- furca direita do dente
    [face]_[numero]_furca_direita_1   <- nível 1 da furca direita
    [face]_[numero]_furca_direita_2   <- nível 2 da furca direita
    [face]_[numero]_furca_direita_3   <- nível 3 da furca direita


Exemplo:

palatina_18
  palatina_18_dente
    palatina_18_coroa
    palatina_18_raiz
  palatina_18_implante
    palatina_18_implante_parafuso
    palatina_18_implante_dente
  palatina_18_furca_esquerda
    palatina_18_furca_esquerda_1
    palatina_18_furca_esquerda_2
    palatina_18_furca_esquerda_3
  palatina_18_furca_direita
    palatina_18_furca_direita_1
    palatina_18_furca_direita_2
    palatina_18_furca_direita_3
